# Django Sticky Notes Application

## Description
This is a Django-based web application that allows users to create, view, update, and delete sticky notes.

---

## Features
- Create notes
- View all notes
- Edit notes
- Delete notes

---

## Technologies Used
- Python
- Django
- SQLite
